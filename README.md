# 介绍
- Demo是一个网站开发的解决方案，它易于使用和部署。对于刚入门的人员来说是一个非常好的方案。它集成了jquery、juicer、众多的工具方法等。对于不使用vue等框架的开发人员，希望本项目能帮助到你。
- 本项目不支持低版本浏览器。
- 本项目不适用于二次开发。

# 安装

## 克隆项目
- git clone https://github.com/sos-luoluo/demo.web

## 安装依赖
- npm install

## 启动服务
- npm run dev
